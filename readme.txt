# Dipesh Subedi - Portfolio Website

A modern, responsive personal portfolio website showcasing my professional profile, technical skills, projects, teaching experience, certifications, and contact information.

🌐 **Live Website:** https://www.subedidipesh.info.np/

---

## About

This portfolio represents my journey as a Software Developer, AI Instructor, and IT Professional. It highlights my educational background, technical expertise, professional experience, and software development projects.

The website is designed to provide visitors with an overview of my work, achievements, and ways to connect with me.

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

2. Move the project into your web server directory (XAMPP/WAMP/LAMP).

3. Import the MySQL database (if applicable).

4. Update the database configuration in the PHP files.

5. Start Apache and MySQL.

6. Open your browser and visit:

```
http://localhost/your-project-folder
```

or

```
https://www.subedidipesh.info.np/
```
